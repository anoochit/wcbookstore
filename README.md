WC Book Store
=============

wcbookstore is book store engine has 2 parts
 1. Mobile Application for mobile reader, manage books and buy book from web book store
 2. Book store is web application contains 3 part
  - User view
  - Seller View
  - Administrator View
 3. Services API is interface for Mobile Application

Tools
=====

 0. LAMPP
 1. Cake PHP
 2. REST Full + JSON Data Format
 3. Google Chart API
 4. Eclipse + ADT
